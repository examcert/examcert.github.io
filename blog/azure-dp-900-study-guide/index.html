Three Months Ago I Couldn't Tell You What a Data Lake Was. Then I Passed Azure DP-900.

Published February 21, 2026 · 5 min read

Three months ago, if you'd asked me the difference between a data warehouse and a data lake, I'd have guessed one was bigger than the other. Then I passed the Azure Data Fundamentals (DP-900) exam with a 920 out of 1000.

This isn't a flex. It's proof that the DP-900 is the most approachable Azure cert Microsoft offers.

Who Is This Cert For?

  • Developers who interact with databases but don't understand data architecture
  • IT professionals exploring data engineering or data science career paths
  • Business analysts who want to speak data language fluently
  • Students who want a Microsoft cert before graduating

What the DP-900 Covers

40-60 questions, 45 minutes:

  1. Core Data Concepts (25-30%) — Relational vs non-relational, OLTP vs OLAP, batch vs streaming
  2. Relational Data on Azure (20-25%) — Azure SQL Database, normalization, basic T-SQL
  3. Non-Relational Data on Azure (15-20%) — Azure Cosmos DB, Blob Storage, Table Storage
  4. Analytics on Azure (25-30%) — Azure Synapse, Data Factory, Power BI, data warehousing concepts

How I Studied (2 Weeks)

Days 1-4: Microsoft Learn's official DP-900 learning path (free, with sandbox labs)

Days 5-8: YouTube reinforcement — Adam Marczak's DP-900 videos

Days 9-12: Practice questions using ExamCert's DP-900 practice tests

Days 13-14: Light review, booked the exam

Things That Tripped Me Up

  • Cosmos DB API Models: Core SQL, MongoDB, Cassandra, Gremlin, Table — know which for which data model
  • Data Factory vs Synapse Pipelines: Both do ETL/ELT, know the distinction
  • Power BI terminology: Datasets, reports, dashboards — the exam cares about these
